# .ENV.LOCAL REQUIREMENTS:
DB_HOST : LocalHost

DB_USER : MySQL Username

DB_NAME : MySQL Database

DB_PASSWORD : MySQL Password

JWT_SECRET : 32 Random Characters (I used https://catsrieldreemurr.github.io/CatsrielsPasswordGenerator/passwords.html)


Keep .env.local file in the root of freshdb.
Run the development server with npm run dev.
Use "npm i" to install dependancies.
